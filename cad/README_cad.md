# /cad — CAD pipeline documentation

All files in this folder are generated by **OpenSCAD 2021.01 +**.
Source files (`.scad`) are parametric — to change a dimension, edit `00_common.scad`
and re-render. To re-render every part:

```bash
cd /cad/scad
for f in 0[1-9]_*.scad 1[0-3]_*.scad; do
    openscad -o "../stl/${f%.scad}.stl" "$f"
done
```

> **Note on the 2-manifold warnings.**  OpenSCAD 2021.01 CGAL emits warnings for the tower lower
> section, the pulley wheel, and the TFT mount because some boolean cuts produce coincident
> faces. **The exported STLs slice cleanly in Cura 5.7 and PrusaSlicer 2.7** — both
> self-heal minor manifold issues. If you ever need a perfectly-watertight STL for hand-checking
> in MeshLab, use `Filters → Cleaning → Repair Non-Manifold Edges`.

## Brass-insert parts list (M3, OD 4.2 mm × 5 mm)

The following parts have heat-set insert pockets — drop in a brass insert with a soldering
iron at 220 °C before final assembly:

- 10_base_plate_section.stl × 6  (5 inserts per panel on the +X edge = 30 inserts)
- 11_pcb_enclosure.stl  (4 inserts in the base for the lid screws)
- **Total: 34 brass inserts** — see BOM line "M3 brass heat-set inserts × 50 (incl. spares)".

## Post-processing per part

| Part | Post-process |
|---|---|
| Towers | Sand mating dovetail (320 grit) for press-fit; cut linear-rail T-slots clean with a small file |
| Deck | Light sand of road grooves; spray-paint road markings if desired |
| Counterweight shells | After printing, fill cavity with **2× M8 hex nut + 4× M8 washer** + 5 g lead fishing weight, weigh on kitchen scale, add/remove until **120 g ± 2 g**, then plug fill hole with PLA glue (Weld-On 3 or super glue) |
| Pulley | Ream Ø6.2 mm bore with 6 mm reamer for clean shaft fit; sand groove with 600 grit so cable runs smoothly |
| Motor mount | Heat-set inserts; press JGA25-370 motor in (light interference fit) |
| Cable guide | Open 1.4 mm slot enough to thread cable through |
| Road barrier | Glue or screw the post to base plate; servo press-fits in pocket |
| PCB enclosure | Heat-set 4× brass inserts in lid screw bosses; fit board on standoffs and tap each for M3 |
| Camera mount | Glue cradle to top of gantry post once cable is threaded; 4× M3 self-tap to hold ESP32-CAM board |
| TFT mount | 4× M3 self-tap holds the TFT module — do not over-tighten; inspect tilt is 35° ± 1° with a digital protractor |

## How a future change is made

1. Edit a constant at the top of `00_common.scad` (say `TOWER_H = 380`).
2. Re-render only the affected parts: `openscad -o ../stl/01_tower_left.stl 01_tower_left.scad`
3. Commit both the updated `.scad` and `.stl` to Git.
